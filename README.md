## 如何使用此專案

```
Windos:
可以打開 Powershell

Mac:
打開 終端機

進入你喜歡的地方，可以放置專案的的地方
```

### 1. 下載專案至你的電腦

> `git clone https://github.com/Fionn88/tender-backend.git`

### 2. 進入目錄

> `cd tender-backend`

### 3. 你有 npm 了嗎？？

> `npm install`
```
如 command not found
請google 搜尋下載 node
```

### 4. 查看你的 node version 是不是 version 17

> `node -v`

```
不一樣沒關係，還是可以跑跑看，能不能啟動.
如不能還請下載至跟我一樣的version
```

### 5. 啟動服務

> `npm start`
